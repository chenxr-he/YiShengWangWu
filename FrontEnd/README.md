1.
安装 Node.js 和 npm：

确保对方已经安装了 Node.js 和 npm。可以通过以下命令检查版本：

```
node -v
npm -v
```
如果没有安装，可以从 Node.js 官网 下载并安装。

2.

进入项目文件夹：


```
cd my-project
```

3.
安装依赖：

在项目根目录下运行以下命令来安装依赖：
```
npm install
```
这个命令会根据 package.json 文件中的内容，安装所有项目依赖项。


4.
运行开发环境：

安装完所有依赖之后，运行以下命令启动开发服务器：

```
npm run dev
```

5.查询具体发送请求的地方

在 my-project\src\components 中 的 WenSHengTU.vue,RenLianFengGeHua.vue,RenLianXiaoXiangHua.vue,ZhiNengKouTu.vue,LiShiJiLu.vue 中查询"具体发送请求"注释即可
